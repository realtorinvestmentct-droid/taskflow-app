# TaskFlow

TaskFlow is a recurring workflow tracker built with React, TypeScript, and Vite.

It is designed to manage bookkeeping-style recurring work with a clean dashboard and separate workspace views.

## Features

- Dashboard for this week’s active tasks
- Task List for upcoming work
- On Hold workflow
- Clients view
- Weekly / Monthly / Quarterly / Annual master views
- Recurring next-date generation
- Browser persistence with localStorage
- Deployed with Vercel

## Tech Stack

- React
- TypeScript
- Vite
- CSS
- localStorage

npm install
npm run dev
